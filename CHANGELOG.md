## [1.0.0] - 2024-12-XX

### المضافة
- دعم تحويل PDF المصور إلى PDF قابل للبحث
- دعم استخراج النص من الصور باستخدام OCR
- دعم متعدد اللغات (العربية والإنجليزية)
- معالجة متوازية لتسريع العملية
- دمج ملفات PDF متعددة
- تحسين الصور قبل OCR لتحسين الدقة
- إعدادات قابلة للتعديل
- دعم macOS و Linux
- اكتشاف تلقائي للأدوات (Tesseract, ImageMagick, Poppler)

### التقنية
- Python 3.7+
- PyPDF2 لمعالجة PDF
- Pillow لمعالجة الصور
- Tesseract OCR لاستخراج النص
- ImageMagick/Poppler لتحويل PDF إلى صور
- ThreadPoolExecutor للمعالجة المتوازية

### الملفات
- `معالج الكتب المصورة.py`: الملف الرئيسي
- `كشكول/01 تحويل إلى قابل للبحث.py`: تحويل PDF
- `كشكول/02 جمع وتقسيم.py`: دمج وتقسيم PDF
- `كشكول/03 تحويل وجمع الصور إلى.py`: تحويل الصور
- `كشكول/04 فتح قفل.py`: إزالة الحماية
