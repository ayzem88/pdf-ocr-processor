# الاختبارات

<div dir="rtl">

هذا المجلد يحتوي على اختبارات الوحدة للمشروع.

</div>

## التثبيت

```bash
# تثبيت متطلبات التطوير
pip install -r requirements-dev.txt
```

## تشغيل الاختبارات

```bash
# تشغيل جميع الاختبارات
pytest tests/

# تشغيل مع تغطية الكود
pytest tests/ --cov=. --cov-report=html

# تشغيل اختبار محدد
pytest tests/test_utils.py
```

## هيكل الاختبارات

- `test_utils.py`: اختبارات الوظائف المساعدة
- `conftest.py`: إعدادات pytest المشتركة

## إضافة اختبارات جديدة

عند إضافة ميزة جديدة، تأكد من إضافة اختبارات لها:

1. أنشئ ملف `test_<اسم_الملف>.py`
2. اكتب اختبارات تغطي الحالات المختلفة
3. تأكد من أن جميع الاختبارات تمر قبل إرسال Pull Request
